# socialnetworkclubdashboard
Social Network Club Dashboard (PyQt6)  A simple desktop application built with Python and PyQt6 that simulates a social network club registration and dashboard. This project is beginner-friendly and demonstrates GUI design, table management, and basic input validation in Python.

Features

User registration with Name, Age, Email, GPA, and favorite game selection.

Data displayed dynamically in a table widget.

Input validation (e.g., age restrictions, GPA checks).

Interactive buttons for submitting, clearing, and exiting the application.

Displays custom messages with QMessageBox.

Integrates images into the interface.

Tech Stack

Python 3.x

PyQt6

Qt Designer for UI layout

Usage

Clone the repository:

git clone https://github.com/yourusername/socialnetwork.git


Install dependencies:

pip install PyQt6


Run the application:

python main.py
